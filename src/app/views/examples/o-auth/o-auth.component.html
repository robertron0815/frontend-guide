oAuth 2.0 for Angular

The OAuth 2.0 authorization framework enables a third-party application to obtain limited access to an HTTP service, either on behalf of a resource owner by orchestrating an approval interaction between the resource owner and the HTTP service, or by allowing the third-party application to obtain access on its own behalf.
Source: IETF Tools

Follow this steps to implement oAuth 2.0 for your custom project.

1. Add oAuth-library to your project by npm or yarn

NPM: npm install angular-oauth2-oidc

YARN: yarn add angular-oauth2-oidc

2. Add Auth-Core folder to your project and edit settings

folder-path: src\app\core

Edit the following settings to your provider: (src\app\core\auth-config.ts)

3. Import functions to your app.module.ts

import OAuthModule and CoreModule to your app.module.ts.
Please have a look in this file for more information.

4. Add and register silent-refresh.html to your project

Copy this file to your project: src\silent-refresh.html

It's important to register this file in your angular.json and check if you can access the file via browser.

5. Add services to your app component

Have a look at this file inside this repository: src\app\app.component.ts. Import the "AuthService" and add the services that are necessary for your custom project.
